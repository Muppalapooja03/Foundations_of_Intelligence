📌 Overview
This repository contains my Day 1 OSINT investigation report on OpenAI, prepared as part of the Intelligence Accuracy / OSINT Track internship. The report demonstrates the application of the intelligence cycle (collection → processing → analysis → dissemination) using only publicly available sources.

🎯 Objectives
Apply OSINT methods to investigate a major AI company.

Structure findings into categories: founding, milestones, corporate structure, funding, products, leadership, legal signals, competitors.

Highlight patterns such as leadership instability, capital dependence, and transparency risks.

Present insights in a professional analyst tone with visual storytelling.

📂 Contents
Report.md / Day1_OpenAI.docx → Full OSINT investigation report.

Visual Annex → Timeline diagrams, org chart, and funding graphs.

Source Mapping → References linked to open sources (Wikipedia, OpenAI.com, CNBC, PitchBook, Tracxn, TechCrunch).

🔎 Intelligence Cycle Applied
Collection: Public sources (company announcements, media, databases).

Processing: Organized into structured categories.

Analysis: Identified risks and recurring themes.

Assessment: Evaluated implications for IPO readiness, governance, and reputation.

Dissemination: Delivered as a structured OSINT dossier.

📊 Key Insights
Capital Position: Strong fundraising, weak profitability.

Leadership Stability: Repeated crises and reshuffles.

Governance Design: Foundation‑controlled PBC balances mission vs. investor demands.

Public Trust Exposure: Transparency controversies present reputational risks.
